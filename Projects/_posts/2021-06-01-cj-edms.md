## CJ대한통운 EDMS [2021.04 - 2021.12]

- EDMS 화면 UI/UX 디자인 설계
- .NET Framework를 활용한 Winform 베이스의 EDMS 구현
- 이미지 뷰어 기능 개선
- IMG->PDF 변환 기능 추가

## 주요 성과
- 사용률 10% 미만이던 기존 시스템을 사용률 95% 이상으로 개선
- 신규 교육 없이 자체 솔루션으로 개발에 성공

## 개발 환경
- **개발 언어**: Java, C#
- **운영체제**: Windows, Linux
- **데이터베이스**: Oracle
- **프레임워크**: Spring Framework, .NET Framework
- **이미지 솔루션**: 자체 이미지 솔루션

## 프로젝트 규모
- 전체 인원: 60명
- 소속 인원: 3명
